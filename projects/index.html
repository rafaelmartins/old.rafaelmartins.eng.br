Projects

These are some of my relevant open source software projects.

A full list of my public Git repositories is available on GitHub: @balde, @rafaelmartins.

balde/balde
Description: A microframework for C based on GLib and bad intentions.
Home page: http://balde.io/
GitHub repository: https://github.com/balde/balde
rafaelmartins/blohg
Description: A Mercurial/Git based blogging engine.
Home page: http://blohg.org/
GitHub repository: https://github.com/rafaelmartins/blohg
rafaelmartins/distpatch
Description: Distfile patching support for Gentoo Linux.
Home page: http://www.gentoo.org/proj/en/infrastructure/distpatch/
GitHub repository: https://github.com/rafaelmartins/distpatch
rafaelmartins/pyoembed
Description: A Python library for oEmbed that supports auto-discovered and manually included providers.
GitHub repository: https://github.com/rafaelmartins/pyoembed
rafaelmartins/dnsimple-dyndns
Description: Dynamic DNS implementation, that relies on DNSimple.com.
GitHub repository: https://github.com/rafaelmartins/dnsimple-dyndns
rafaelmartins/cgisysinfo
Description: A small cgi utility to show basic system information on linux machines.
GitHub repository: https://github.com/rafaelmartins/cgisysinfo
rafaelmartins/ownpaste
Description: Private pastebin (server-side implementation).
Home page: http://ownpaste.rtfd.org/
GitHub repository: https://github.com/rafaelmartins/ownpaste
rafaelmartins/github-repo-mirror
Description: A flask app that listens to Github webhooks and fetches latest changes to a local repository.
GitHub repository: https://github.com/rafaelmartins/github-repo-mirror
rafaelmartins/github-pages-publish
Description: A script that commits files from a directory to the gh-pages branch of the current Git repository.
GitHub repository: https://github.com/rafaelmartins/github-pages-publish
rafaelmartins/g-octave
Description: A tool that generates and installs ebuilds for Octave-Forge packages "on-the-fly" for Gentoo Linux.
GitHub repository: https://github.com/rafaelmartins/g-octave
